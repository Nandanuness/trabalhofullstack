Consttução de um cadastro de pessoas físicas, obrigatório dados: pessoa, nome, CPF, telefone.

Frontend: HTML e CSS.
Backend: PHP.
Banco de Dados: MYSQL.


Para rodar o projeto no terminal, utilizei **php -S localhost:3000**
